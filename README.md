# CSI Arduino

Sitio web del programa **Principios de Arduino** del Cuerpo de Solidaridad Informática (CSI) — MEDUCA, Panamá.

Programa educativo de Daniel Abadi que introduce a estudiantes de escuelas públicas en electrónica, programación y pensamiento lógico a través de 12 talleres con Arduino.

## Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo local

```bash
npm install
npm run dev
```

Abre http://localhost:3000.

## Despliegue

Conectado a Vercel — cada push a `main` despliega a producción automáticamente.
